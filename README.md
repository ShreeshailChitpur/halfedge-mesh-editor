# HalfEdge Mesh Editor

A C++ OpenGL-based mesh editor implementing a custom Half-Edge data structure for mesh representation and manipulation.

## Features

* OpenGL rendering pipeline
* GLFW window and input handling
* Camera and viewport system
* Half-Edge mesh data structure
* Triangle and line rendering
* Basic mesh transformations
* Modular project architecture

## Tech Stack

* C++
* OpenGL
* GLFW
* GLM

## Project Structure

```text
.
├── MeshEditor/        # Main application
├── GLRenderSystem/   # Rendering system
├── HalfEdge/         # Half-edge data structure
├── Interfaces/       # Shared interfaces
└── ThirdParty/       # External libraries
```
